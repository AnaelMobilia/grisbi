GtkWidget* gsb_assistant_new ( const gchar *  title,
const gchar *  explanation,
gchar *  image_filename,
GCallback  enter_callback 
)

Create and initialize a new grisbi assistant. It is basically composed of a GtkDialog with a notebook that is switched from page to page when user click on dialog buttons.

Paramètres:
title Title of the assistant.
explanation Short text to display in the first page of the assistant.
image_filename Icon to display in the title. (if NULL, use grisbi.png, default logo)
enter_callback A callback to connect to the "switch-page" callback when go to the first page of the Grisbi assistant notebook. (the callback should be : gboolean callback ( GtkWidget *assistant, gint new_page ) )
