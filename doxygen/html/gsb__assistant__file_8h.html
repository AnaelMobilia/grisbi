this function is called to launch the file opening assistant

Paramètres:
first_opening TRUE if this is the first opening of Grisbi, and that assistant is continuing from the general configuration assistant
import TRUE if we come from import (possible when we do file->import without anything opened, because we need to set up the variables and currencies)
Renvoie:
a GtkResponseType containing the return value at the end of the assistant
