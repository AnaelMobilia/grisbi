GtkWidget* utils_font_create_button ( gchar **  fontname,
GCallback  hook,
gpointer  data 
)

create a button displaying the name and the size of the font clicking on that button show a GtkFontSelectionDialog to change the font and update the fontname !! the fontname can be changed and freed so *fontname must not be const

Paramètres:
fontname a pointer to a pointer to the name of the font to change with the selection of the user
hook An optional function to execute when the font has changed (hook must be func ( gchar new_font_name, gpointer data ) )
data An optional pointer to pass to hooks.

a GtkButton
