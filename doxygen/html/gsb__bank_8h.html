create a combo_box with the name of all the banks and an 'add' button at the end update too if filled the value of the associated account (see gsb_autofunc_... for more explains)

Paramètres:
bank the index we want to place the combobox
hook an optional function to execute if the combobox is modified
data an optional pointer to pass to hook
default_func a function to change the value in memory (should be gsb_data_account_set_bank or NULL)
number_for_func the gint to send to the default_func (should be the account number)
Renvoie:
GtkWidget * the combo_box
