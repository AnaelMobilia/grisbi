GtkWidget* gsb_automem_checkbutton_new ( const gchar *  label,
gboolean *  value,
GCallback  hook,
gpointer  data 
)

Create a GtkCheckButton with a callback associated. Initial value of this checkbutton is set to the value of *value. This checkbutton calls gsb_automem_checkbutton_changed upon toggle, which in turn modifies *data. If a hook is possibly executed as well.

Paramètres:
label The label for this checkbutton
value A pointer to a boolean integer
hook A GCallBack to execute if not null (hook must be : func ( GtkWidget *toggle_button, gpointer data) )
data An optional pointer to pass to hook
Renvoie:
A newly allocated GtkCheckButton

Creates a new GtkSpinButton with a pointer to a string that will be modified according to the spin's value. this is a gint spin button this function is the same as gsb_automem_spin_button_new_full, but set defaults parameters by default : between 0 and G_MAXDOUBLE ; value is integer, step increment is 1 and page is 5
