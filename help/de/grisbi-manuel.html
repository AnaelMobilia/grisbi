Sie werden nur einige Nachrichten pro Jahr erhalten - neue Versionen und Bugfixes.

Seit der Version 0.4.1 ist das Handbuch nicht mehr ein fixer Bestandteil der Grisbi Basisinstallation. Die Gründe dafür sind, das Grisbi
laufend weiterentwickelt wird und die Mehrsprachigkeit.

Verfügbare Versionen

Das Handbuch ist in zwei Versionen verfügbar:
  • Eine reine Textversion;
  • eine illustrierte Version mit vielen Bildschirmphotos.
Die zweite Version ist zweifelsfrei benutzerfreundlicher - jedoch dauert der Download länger.

Das Handbuch installieren

Auf der Grisbi Webseite Grisbi's site finden Sie das Handbuch im Bereich Documentation auf der Download Seite. Derzeit ist nur die französische Version Verfügbar. Wenn Sie das Handbuch in ihrer Sprache übersetzen wollen, senden Sie bitte ein eMail an:
Daniel

Ein Paktet installieren

Wie das Programm selbst, ist das Handbuch ebenfalls als Debian oder RPM Pakte verfügbar.
Wenn Ihre Distribution keine Pakteinstallation unterstützt, lesen sie bitte den Abschnitt 'Manuelle Installation'
Die Dateien des Handbuches werden im Verzeichnis /usr/share/doc/grisbi/help/de installiert.

Manuelle Installation

Wenn eine Paktetinstallation nicht möglich ist, oder wenn Sie es bevorzugen das Handbuch manuell zu installieren, müssen Sie
zuerst das Handbuch [manuel-grisbi-xxx-n.n.n.tar.gz] herunterladen. xxx steht für die Version (Text oder illustiert), n.n.n ist die
Versionsnummer des Handbuches.
Entpacken Sie das Archiv und kopieren Sie alle .html, .png und die topic.dat Datei in das Verzeichnis:
/usr/share/doc/grisbi/help/de.
