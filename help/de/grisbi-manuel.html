Haben Sie sich schon in die Grisibi Infos Mailingliste eingetragen?

Sie erhalten nur einige Nachrichten pro Jahr, speziell über neue Versionen und Fehlerbehebungen.

Seit der Version 0.4.1 ist das Handbuch kein fixer Bestandteil der Grisbi Basisinstallation.
Die Gründe dafür sind, das Grisbi laufend weiterentwickelt wird und die Mehrsprachigkeit.

Verfügbare Versionen

Das Handbuch ist in zwei Versionen verfügbar:
  • Eine reine Textversion;
  • eine illustrierte Version mit vielen Bildschirmfotos.
Die zweite Version ist zweifelsfrei benutzerfreundlicher - jedoch dauert der Download länger.

Das Handbuch installieren

Auf der Grisbi Webseite finden Sie das Handbuch im Bereich Dokumentation auf der Download Seite.
Derzeit ist nur die französische Version verfügbar. Wenn Sie das Handbuch in Ihre Sprache übersetzen wollen,
senden Sie bitte ein eMail an: Daniel

Ein Paktet installieren

Wie das Programm selbst ist das Handbuch ebenfalls als Debian oder RPM Paket verfügbar.
Wenn Ihre Distribution keine Pakteinstallation unterstützt, lesen Sie bitte den Abschnitt 'Manuelle Installation'.
Die Dateien des Handbuches werden im Verzeichnis /usr/share/doc/grisbi/help/de installiert.

Manuelle Installation

Wenn eine Paketinstallation nicht möglich ist, oder wenn Sie es bevorzugen das Handbuch manuell zu installieren, müssen Sie
zuerst das Handbuch [manuel-grisbi-xxx-n.n.n.tar.gz] herunterladen.
xxx steht für die Version (Text oder illustiert), n.n.n ist die Versionsnummer des Handbuches.
Entpacken Sie das Archiv und kopieren Sie alle .html, .png und die topic.dat Datei in das Verzeichnis: /usr/share/doc/grisbi/help/de.
