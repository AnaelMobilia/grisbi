Chapitre 12   Les menus

Les menus de Grisbi sont assez intuitifs mais nous allons malgré tout les passer rapidement en revue.

12.1   Menu Fichier

C'est le menu le plus classique et celui de Grisbi ne déroge pas à la règle. Vous y trouverez les choix suivants :

  • Nouveau : crée un nouveau fichier de compte. Attention ce n'est pas la même chose que créer un nouveau compte ;

  • Ouvrir : raccourci clavier <F3> ;

  • Enregistrer : raccourci clavier <Ctrl><S> ;

  • Enregistrer sous... : Grisbi vous propose par défaut le répertoire courant avec l'extension .gsb ;

  • Derniers fichiers : affiche une liste des n derniers fichiers ouverts avec Grisbi, n étant configurable par le menu Configutation/Fichiers (voir le paragraphe 13.2) ;

  • Importer : affiche une liste des différents formats importables. Pour l'instant seul le format QIF est supporté par Grisbi, mais le format OFX est prévu pour une future version ;

  • Exporter : mêmes remarques que pour l'option Importer ;

  • Fermer : raccourci clavier <Ctrl><W> ;

  • Quitter : raccourci clavier <Ctrl><Q>.

12.2   Menu Compte

Toutes les fonctions relatives aux comptes :

  • Nouveau compte : bascule dans l'onglet Opérations, et affiche les propriétés du compte Sans nom. Il ne vous reste plus qu'à compléter ;

  • Supprimer un compte : ouvre une boîte de dialogue de choix du compte à supprimer (par défaut celui qui est utilisé dans l'onglet Opérations). Attention, cette opération est irréversible!

  • Opérations du compte : affiche les propriétés du compte. Vous pouvez aussi utiliser l'icône de droite de l'onglet Opérations ;

  • Compte cloturé : affiche la liste des comptes cloturés et permet de les ouvrir de nouveau (voir le paragraphe 3.3).

12.3   Menu Configuration

Ce menu reprend toutes les options de la fenêtre de configuration et ouvre celle-ci sur l'onglet correspondant.

  • Général ;

  • Fichier ;

  • Echéances ;

  • Affichage ;

  • Appelette ;

  • Devises ;

  • Banques ;

  • Exercices ;

  • Types d'opération.

12.4   Menu Aide

Menu standard également, comprenant :

  • Index des rubriques : affiche l'aide que vous lisez en ce moment ;

  • Démarrage rapide : affiche un texte d'aide très succint pour les impatients ;

  • A propos... : affiche la classique boite de dialogue du même nom.
