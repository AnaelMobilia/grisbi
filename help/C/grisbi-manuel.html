First of all, have you subscribe to Grisbi's infos mailing list?

You will receive only a few messages a year, about the new versions and the bugs fixes.

So don't wait, do it just now!

Since the 0.4.1 version, the user guide is no more integrated in the package of Grisbi. Effectively, its ever-growing weight and its internationalisation obligend us to create a separate package.

The guide types

The user guide comes in two types:
  • a text only type;
  • an illustrated type, with many screenshots.
It is obvious that the second one is more pleasant to use but also more long to download. So it's up to you...

Installing the guide

On Grisbi's site you'll find the package you need in the section Dowload/Documentation. For the moment only the french version is avaliable. Translators welcome! Send a mail to Daniel if you want to help Grisbi.

Installing a package

As well as the programm, the guide is avaliable as packages deb or rpm. However there are no specifics packages for the differents distributions. Install tha package as you use to. The files will be placed in /usr/share/gnome/help/grisbi/C.

Installing by hand

If for any reason you need (or prefere) process the installation by hand, you must download the archive manuel-grisbi-xxx-n.n.n.tar.gz where xxx is the version (text or illustrated, txt or img respectively) and n.n.n the version. Untar this archive and copy all the .html and .png files as well as the topic.dat in /usr/share/gnome/help/grisbi/C.
